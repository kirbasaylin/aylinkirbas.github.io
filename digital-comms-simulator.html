Results that check out

The point isn't that it runs — it's that the measured curves land exactly where the math says they should. Every figure is real simulator output.

FIG.01BER vs Eb/N0. Monte-Carlo points sit on the closed-form curves for all four schemes. BPSK and QPSK overlap exactly — the correct, expected result for Gray-coded systems, not a bug.
FIG.02The 16-QAM waveform, end to end. Ideal Gray-coded constellation, the received cloud after matched filtering, the root-raised-cosine transmit spectrum, and the three-level eye diagram — a clean eye means the pulse shaping adds no inter-symbol interference.
FIG.03What fading costs you. The steep AWGN waterfall collapses into a slow Rayleigh decay once the channel fades — and the simulation tracks the analytic Rayleigh curve, quantifying why wireless links need far more power, or diversity, for the same reliability.

Under the hood

The constellation is built from the Gray-code construction rather than a qammod call, so neighbouring symbols always differ by a single bit. Detection is the matched minimum-distance rule, provably optimal for square QAM in Gaussian noise.

Pulse shaping uses a self-contained root-raised-cosine filter; cascading it at transmitter and receiver yields the zero-ISI Nyquist response visible in the eye diagram. Noise is referenced to a target Eb/N0, and the theoretical BER uses the Gaussian Q-function written through erfc — so the whole thing runs on base MATLAB, no toolboxes.

link_example.mMATLAB
% bits → 16-QAM → noisy channel → bits
bits = randi([0 1], 1, 10000);
tx   = qam_mod(bits, 16);
rx   = awgn_channel(tx, 12, 4);  % Eb/N0 = 12 dB
bhat = qam_demod(rx, 16);
ber  = mean(bhat ~= bits)      % measured BER

% does it match theory? (it does)
theory = theory_ber(16, 12);

Run it yourself

Clone the repo and open the folder in MATLAB (or GNU Octave — it runs there too).
Type main. It adds the library to the path, seeds the RNG for reproducibility, and runs all three demos.
Find every figure regenerated in the results/ folder — the same ones shown above.
